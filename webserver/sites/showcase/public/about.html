About This Showcase

This example site demonstrates how to structure your projects for 3KPRO Webserver.

Site Structure

The site follows this structure:

  • sites/showcase/public/ - Static files
  • sites/showcase/app.js - Dynamic Express router

How It Works

The webserver automatically:

  • Serves static files from /public folder
  • Mounts Express routes from app.js
  • Provides a clean URL structure

Technologies

This showcase demonstrates:

  • HTML5, CSS3 and JavaScript
  • Express.js for API endpoints
  • Responsive design principles
  • Fetch API for AJAX requests
